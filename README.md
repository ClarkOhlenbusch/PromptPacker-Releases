# PromptPacker

PromptPacker is a powerful, cross-platform tool designed to optimize and compress prompts for use with language models. It enables users to maximize token efficiency while maintaining the integrity and meaning of their input, making it an essential utility for developers, researchers, and AI enthusiasts working with token-limited APIs and applications.

## Overview

PromptPacker helps you reduce the token count of your prompts through intelligent compression techniques. By optimizing your prompts before sending them to language models, you can:

- **Reduce costs** associated with token-based API pricing
- **Improve performance** by fitting more context within token limits
- **Maintain quality** while achieving significant compression ratios
- **Optimize workflows** across multiple platforms seamlessly

## Key Features

### Token Compression
PromptPacker employs sophisticated algorithms to compress prompts without sacrificing semantic meaning:
- Intelligent whitespace optimization
- Redundancy elimination
- Smart abbreviation techniques
- Contextual preservation for accurate model interpretation

### Cross-Platform Support
PromptPacker is fully compatible across all major operating systems:
- **Windows** - Full support with native installers
- **macOS** - Universal binaries for Intel and Apple Silicon
- **Linux** - Compatible with major distributions
- Consistent user experience and functionality across all platforms

### File Handling
Flexible file operations to suit your workflow:
- **Multiple input formats** - Support for .txt, .md, and other text-based formats
- **Batch processing** - Process multiple prompts in one operation
- **Output customization** - Choose your preferred output format and destination
- **Encoding support** - Automatic handling of various text encodings
- **Clipboard integration** - Direct copy/paste support for quick operations

### Additional Capabilities
- Real-time token counting and compression statistics
- Preview compressed output before saving
- Preserve formatting options for specific use cases
- Customizable compression profiles
- History tracking for recent operations

## Getting Started

### Download Instructions

You can download the latest version of PromptPacker from the [Releases page](https://github.com/ClarkOhlenbusch/PromptPacker-Releases/releases).

#### Platform-Specific Downloads

**Windows:**
- Download the `.exe` installer for automatic installation
- Alternatively, download the portable `.zip` version for no-install usage
- Extract and run directly (portable version)

**macOS:**
- Download the `.dmg` file for Intel Macs
- Download the `.dmg` file for Apple Silicon (M1/M2/M3) Macs
- Open the `.dmg` file and drag PromptPacker to Applications folder

**Linux:**
- Download the `.AppImage` file for universal Linux compatibility
- Make the file executable: `chmod +x PromptPacker.AppImage`
- Run directly: `./PromptPacker.AppImage`
- Alternative: Download the `.tar.gz` archive and extract to your preferred location

### Installation

1. Navigate to the [Releases page](https://github.com/ClarkOhlenbusch/PromptPacker-Releases/releases)
2. Find the latest release version
3. Download the appropriate file for your operating system
4. Follow the platform-specific installation instructions above
5. Launch PromptPacker and start optimizing your prompts

### Quick Start

1. **Open PromptPacker** - Launch the application on your platform
2. **Load a prompt** - Paste text, load a file, or use clipboard content
3. **Configure settings** - Adjust compression level and output preferences
4. **Compress** - Click the compress button to optimize your prompt
5. **Review results** - Check the token count reduction and preview
6. **Save or copy** - Export the compressed prompt to file or clipboard
